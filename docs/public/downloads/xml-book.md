# XML: Fahrtenbuch

Im Fahrtebuch werden Autofahrten eingetragen. Folgende Daten werden gespeichert:

- Kennzeichen
- Startzeit
- Endezeit
- Startdatum
- Enddatum
- Start-km
- End-km
- Startpunkt
- Endpunkt
- Beschreibung
